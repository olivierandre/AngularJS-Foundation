Project AngularJS + Foundation
====================

Create with Yeoman.
    -   HTML5
    -   CSS3
        -   Framework 'Foundation'
    -   Javascript
        -   AngularJS
    -   BDD
        -   MongoDB

### node_modules
```
npm install
```
### librairies (Angular, Bootstrap, jQuery, font-awesome, animate.css ...)
```
bower install
```

### Launch server
```
grunt serve --force
```
